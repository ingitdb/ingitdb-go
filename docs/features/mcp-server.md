# MCP server for inGitDB

## Skills

- List known inGidDB databases registered in the `~/.ingitdb/ingitdbs.yaml`
- List inGitDB objects (metadata):
    - Collections
    - Materialized views
    - Subscribers
- Provide metadata about a collection (_fields, triggers, related views, etc._)
- Get records from an inGitDB collection
- CRUD operations with a collection
