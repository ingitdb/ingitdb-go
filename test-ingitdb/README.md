# test-ingitdb

This demo inGitDB provides examples of how to use inGitDB for version control and collaboration.

It includes sample data and explanations to help users understand and implement inGitDB in their projects.

It is also used in tests to validate functionality and ensure compatibility with different use cases.

## Examples of the collections

- todo
    - tasks
    - tags
    - statuses
- geo
    - countries
        - cities
